# Function: hashPrompt()

> **hashPrompt**(`prompt`): `string`

Hash a prompt string with SHA-256.
Produces "sha256:&lt;full-hex&gt;" format matching provider-http hashPromptFull.
The raw prompt is never retained.

## Parameters

| Parameter | Type |
| ------ | ------ |
| `prompt` | `string` |

## Returns

`string`
