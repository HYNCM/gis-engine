# @gis-engine/scene3d

`@gis-engine/scene3d` remains an experimental contract package in v1.4.0.

## What Lives Here

- `SceneView3DExtensionSchema`
- SceneView3D preparation commands
- mock snapshot/query contracts
- Scene resource load-plan validation and release-gate helpers

## Status

- Public package boundary: available
- Stable `view.mode: "scene3d"` runtime: blocked
- Real renderer promotion: not accepted in this sprint

## How To Use This Page

- Use this overview for boundary and status.
- Use package source and repo review evidence for promotion context.
- Do not read API presence here as evidence of stable 3D runtime support.
