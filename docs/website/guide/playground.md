# Playground

Try GIS Engine directly in your browser — no installation required.

## Basic GeoJSON Map

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/HYNCM/gis-engine/tree/main/examples/basic-geojson)

This playground loads the basic-geojson example with:

- `@gis-engine/engine` SDK
- MapLibre GL renderer
- Sample GeoJSON data points

## What You Can Do

1. Edit `src/main.ts` to change the map spec
2. Add layers, change the view, or modify styles
3. See changes in real-time
