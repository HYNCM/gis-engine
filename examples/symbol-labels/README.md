# Symbol Labels Example

Demonstrates GIS Engine's **full symbol layer type** with text and icon annotations.

## What it shows

- 12 world cities as labeled markers
- Zoom-dependent `text-size` and `icon-size` via `interpolate`
- `match` expression for color-coding by city type (megacity, capital, major)
- Text halos, text-anchor positioning, and overlap control
- `text-allow-overlap` and `icon-allow-overlap` for decluttering

## Run

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal (usually `http://localhost:5173`).
